Common DTC trouble codes explained

June 23, 2026 · 8 min read · Diagnostics

DTC codes can seem like alphabet soup at first, but they follow a logical pattern. Understanding the structure and learning the most common codes will significantly speed up your diagnostic work and help you explain issues to customers clearly.

DTC structure: What the letters and numbers mean

Every trouble code follows the format: P/B/C/U followed by four digits.

  • P = Powertrain (engine, transmission, hybrid systems)
  • B = Body (doors, windows, lighting, seats)
  • C = Chassis (steering, suspension, ABS, braking)
  • U = Undefined/Network (communication, CAN bus faults)

The first digit after the letter indicates severity: 0 = OBD standard, 1 = Manufacturer specific.

💡 Example: P0420 (Catalyst System Efficiency Below Threshold) starts with P0, so it's a standardized powertrain code. P1234 would be manufacturer-specific.

Top 10 powertrain codes you'll see constantly

P0300 – Random/Multiple Cylinder Misfire

  • Common causes: Spark plugs, ignition coils, fuel injector, compression issue
  • Diagnosis approach: Check fuel trims, monitor misfire counters, perform compression test
  • Quick fix check: Visual inspection of plugs and coil packs first

P0171/P0174 – System Too Lean (Bank 1/2)

  • What it means: Engine is running with too much air, not enough fuel
  • Common culprits: Vacuum leak, failing MAF sensor, weak fuel pressure
  • Check: Fuel pressure graph, vacuum hoses, MAF sensor cleanliness

P0420 – Catalyst System Efficiency Below Threshold

The famous "check engine light" that keeps coming back.

  • Usually means: Catalytic converter efficiency drop OR an underlying issue (above codes unresolved)
  • Key insight: Fix the real problem first—misfire, lean condition, etc. The cat code is often a symptom
  • Don't just replace the cat: That'll cost you credibility and money

P0128 – Coolant Temperature Not Reaching Thermostat Regulating Temperature

  • Common causes: Stuck-open thermostat, failed coolant temperature sensor, weak radiator fan
  • Quick check: Monitor live coolant temps during warm-up—should reach setpoint

P0401 – EGR Flow Insufficient

  • Common in: Older vehicles, high-mileage engines
  • Causes: Carbon buildup, EGR valve stuck, exhaust restriction
  • Diagnosis: Check EGR duty cycle during live data, vacuum test

P0133 – Oxygen Sensor Circuit Slow Response

  • What it means: Oxygen sensor isn't switching fast enough—might not be bad yet
  • Common causes: Aging sensor, fuel trim issues, spark plug condition
  • Don't rush: Check fuel trims and long-term fuel trim first

P0505 – Idle Control System Malfunction

  • Causes: Vacuum leak, carbon buildup in idle air control, MAF sensor issue
  • Diagnosis: Record live idle speed data, check for vacuum leaks with smoke test

P0706 – Transmission Range Sensor Circuit

  • Transmission issue: Vehicle may not shift properly or start in gear
  • Check: Park switch continuity, manual linkage adjustment

P0430 – Catalyst System Efficiency Below Threshold (Bank 2)

Same as P0420 but on the opposite cylinder bank. Same advice applies.

P0440 – Evaporative Emission Control System Malfunction

  • Common in: Vehicles with faulty fuel caps or charcoal canister issues
  • Often: A loose gas cap—tell the customer to tighten it and clear the code
🔧 Pro tip: Always pull freeze frame data with the code. It tells you engine load, fuel trims, and other parameters when the fault occurred—invaluable for diagnosis.

The golden rule: Don't code-chase

Reading a code is the starting point, not the diagnosis. Use the code as a clue, then investigate:

  • Check live data to understand what's really happening
  • Review the freeze frame—when did the fault occur?
  • Perform targeted tests based on the system, not just the code
  • Some codes are symptoms (P0420) of other issues; fix the root cause
