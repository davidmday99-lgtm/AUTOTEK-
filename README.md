# AUTOTEK

Professional Texa diagnostic tools — St. Louis, MO & Minneapolis, MN. Ships all 50 states.

## Products
- **Axone Voice** — AI-powered diagnostic tablet with voice commands
- **Deluxe Fleet Package** — Complete fleet diagnostic solution
- **E Truck** — Electric vehicle & truck diagnostics
- **AC Service (Konfort)** — Professional AC service equipment

## Tech Stack
- Static HTML/CSS/JS (no frameworks)
- GitHub Pages for hosting
- Formspree for lead capture
- GitHub Actions for daily blog automation

## Setup
1. Enable GitHub Pages: Settings > Pages > Deploy from branch `main`, folder `/ (root)`
2. Formspree form ID is configured in `contact.html`
3. Daily blog posts are generated automatically via `.github/workflows/daily-blog.yml`
4. Custom domain: Settings > Pages > Custom domain (when ready)
